HashMap
============

A simple HashMap with collision management written in Java.

Usage
---------
```java
HashMap<String, Integer> map = new HashMap<>();
map.put(null, 0);
map.put("key", 1987);

if ( map.hasElement("key") ) {
	Integer value = map.get("key");
}
```

How To Get Started 
---------
1. Just drag & drop HashMap.java into your project.
2. That's all!
